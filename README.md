# 🌍 多时区数字时钟

一个现代化的、功能丰富的多时区数字时钟 Web 应用，实时显示世界各地的时间。

![License](https://img.shields.io/badge/license-MIT-green)
![Version](https://img.shields.io/badge/version-1.0.0-blue)

## ✨ 功能特点

### 核心功能
- 🕐 **实时时钟显示** - 每秒更新多个时区的时间
- 🌐 **全球 30+ 时区** - 覆盖世界主要城市和地区
- 🔍 **智能搜索** - 快速搜索时区（支持城市名、时区代码、地区等）
- 💾 **本地存储** - 自动保存选择的时区配置
- ⚡ **即时切换** - 快速添加和移除时区

### 显示信息
- **24小时制时间** - 精确到秒
- **12小时制时间** - 方便对比
- **日期** - 当地日期
- **星期几** - 中文显示
- **时区偏移** - UTC 相对时间
- **地区信息** - 城市所属地区

## 🎨 设计特色

- 🌙 **深色主题** - 护眼设计
- 📱 **完全响应式** - 支持桌面、平板、手机
- ✨ **现代化界面** - 流畅的动画和过渡效果
- 🎯 **直观交互** - 简单易用的操作

## 🚀 快速开始

### 安装
1. 克隆或下载项目
```bash
git clone https://github.com/xiaoguangtang1-a11y/digital-clock.git
cd digital-clock
```

2. 打开浏览器
直接打开 `index.html` 文件，或使用本地服务器：
```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js (http-server)
npx http-server
```

3. 访问应用
打开浏览器访问 `http://localhost:8000`

## 🎯 使用指南

### 添加时区
1. 点击 "+添加时区" 按钮
2. 在搜索框中搜索城市或时区名称
3. 点击要添加的时区

### 移除时区
1. 将鼠标悬停在时钟卡片上
2. 点击右上角的 × 按钮

### 重置
点击 "↻重置" 按钮恢复默认的三个时区：
- 北京 (中国)
- 伦敦 (英国)
- 纽约 (美国)

### 搜索时区
支持多种搜索方式：
- 城市名：北京、Tokyo、New York
- 国家名：China、Japan、USA
- 时区代码：Asia/Shanghai
- 地区名：亚洲、Europe

- **平板** (481-768px) - 2列网格

选择的时区会自动保存到浏览器的 LocalStorage，刷新页面后配置会保留。

## 🔧 配置

### 修改默认时区
编辑 `script.js` 中的 `DEFAULT_TIMEZONES` 数组：
```javascript
const DEFAULT_TIMEZONES = ['Asia/Shanghai', 'Europe/London', 'America/New_York'];
```

### 添加新时区
在 `TIMEZONES` 数组中添加新条目：
```javascript
{ 
    name: '城市名 (国家)', 
    timezone: 'Continent/City', 
    region: '地区', 
    offset: 'UTC±X' 
}
```
